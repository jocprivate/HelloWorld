# Hello World

This application has one business requirement - writing "Hello World" To The Screen

The application consists of Web API as the Middle Tier and HTML and jQuery as the Front End.  

Majority of the code is method signatures that provide a skeleton for creating, inserting, updating, and deleting greetings. There is full implementation of The Get Method in the GreetingController.   

A Unit Tests project is also part of the solution.

Technology Stack: Web API 2, HTML, jQuery
Requirements: Visual Studio 2015

Getting Started
1) Build and Run The Greetings.sln File (Application should run using IISExpress)
2) Under The Folder Greeting.UI, open the file index.html in your favorite browser
3) Hello World should be displayed in a table
